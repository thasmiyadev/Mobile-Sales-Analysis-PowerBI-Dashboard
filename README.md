# Mobile-Sales-Analysis-PowerBI-Dashboard
Interactive Mobile Sales Performance Dashboard built using Microsoft Power BI to analyze sales, transactions, customer ratings, payment methods, and brand performance.

# Project Overview

This project is an interactive **Mobile Sales Performance Dashboard** developed using **Microsoft Power BI**. The dashboard provides comprehensive insights into mobile sales performance, customer behavior, brand performance, payment methods, and sales trends through interactive visualizations.

The dashboard enables business users to monitor KPIs, identify high-performing brands, analyze customer ratings, and make data-driven decisions.

---

# Business Objectives

- Monitor overall mobile sales performance
- Analyze monthly sales trends
- Compare sales across different mobile brands
- Evaluate payment method preferences
- Analyze customer ratings
- Identify city-wise sales distribution
- Monitor daily sales performance
- Track transaction volume

---

# Key Performance Indicators (KPIs)

- 📈 Total Sales
- 📦 Total Quantity Sold
- 💳 Total Transactions
- 💰 Average Sales
- ⭐ Customer Ratings
- 📍 City-wise Sales

---

# Dashboard Features

### Executive Dashboard

- KPI Cards
  - Total Sales
  - Total Quantity
  - Total Transactions
  - Average Sales

- Interactive Slicers
  - Month
  - Mobile Model
  - Brand
  - Payment Method

- Interactive Visualizations
  - Sales by City (Map)
  - Quantity by Month (Line Chart)
  - Sales by Mobile Model (Column Chart)
  - Transactions by Payment Method (Pie Chart)
  - Customer Rating Distribution (Funnel Chart)
  - Sales by Day Name (Area Chart)
  - Brand Performance Table
